# chatrooms

A Chatroom API using the Gin framework.

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/rateitorg/chatrooms && cd chatrooms
```

### 2. Run API

```bash
go run main.go
```

## Usage

### Version Endpoint

```bash
curl http://127.0.0.1:8080/version
```

#### Example Response

```json
{
  "name": "Chatroom WebSocket API",
  "buildDateTime": "4/12/2024 0:49:24",
  "version": "0.0"
}
```
